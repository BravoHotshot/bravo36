# Bravo36
Custom firmware for my wacky split keeb with arch36 layout but corne style board (Caterina bootloader with ProMicro board)

Based on crkbd_rev1

# Installation
1. Download the pre-compiled hex from Releases
2. Connect one half in DFU mode
3. Flash the firnware hex using QMK Toolbox
4. Repeat for the other half

# I don't like this
Follow the same steps to flash another firmware.

The steps to enter DFU are the same as crkbd_rev1 -

  Hold Q for left half and P for right half when connecting to the PC to go into DFU. Keep the button held until the flash is complete.
